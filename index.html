// our story

Why We Built This

In our 3rd year at ICAM Strasbourg-Europe, we had a grand total of 6 hours of lab time across 2 sessions to discover industrial robotics — working with a Stäubli 4-axis SCARA and a 6-axis arm. With 20 students and only 2 robots per group, we spent most of the time watching. When we finally got to the controls, safety stops triggered constantly — each one requiring the teacher to intervene, while the same teacher was already fielding a queue of questions from 19 other students. During one session, a finger on the 6-axis gripper snapped off mid-use — the programs we'd spent hours writing became impossible to test, and nobody could fix it on the spot. On top of all that, the robots ran on a proprietary language we'd never encounter again outside that lab. And this is how we decided to pursue EduBot as our 4th-year technical project.

// about

The Problem with Robotics Education

Industrial robotic arms cost €13,000–€25,000 — far beyond the reach of most schools and universities. The gap between classroom and industry remains wide, and students rarely get hands-on access to capable hardware.

Existing educational alternatives are either too limited — few degrees of freedom, inadequate reach — or fragmented, requiring students to juggle separate platforms for theory, coding, and hardware control.

EduBot bridges this gap: a capable 5-axis robotic arm paired with a unified learning ecosystem, all within a Bill of Materials of €620.

// the robot

Technical Overview

A 5-axis serial kinematic chain built from the ground up for educational environments.

01

Mechanical

5-axis serial kinematic chain. J1 base rotation (360°), J2/J3 shoulder and elbow with cycloidal gear reducers for torque, J4/J5 wrist pitch and roll, plus servo-driven parallel-jaw gripper. Structural links use 2020 aluminium extrusion profiles.

5 axes cycloidal drive Al 2020
02

Electrical & Control

Custom in-house PCB with Arduino Mega microcontroller. Five DMT320T stepper motor drivers with rotary encoders at each joint for closed-loop position control. 24 V DC power supply — fully safe for classroom environments.

custom PCB Arduino Mega 24V DC
03

Learning Platform

Next.js web application with Blockly visual editor for beginners and Python text editor for advanced users. Includes 3D digital twin, structured course modules, and ROS 2 communication — all in one interface.

Next.js Blockly ROS 2

Engineering Highlight

Custom Cycloidal Gear Reducers

Designed entirely in-house. 20:1 reduction ratio delivering ~7.7 N·m output torque at joints J2 and J3 — enabling the arm's reach and payload capability at a fraction of the cost of commercial reducers.

// learning platform

One Platform. Everything You Need.

From beginner visual programming to advanced Python — the full learning journey in a single web interface.

01

Blockly Visual Editor

Drag-and-drop programming blocks for beginners. No prior coding experience required — ideal for secondary school students encountering robotics for the first time.

02

Python Text Editor

Full Python programming environment for advanced users, CS students, and educators building custom curricula and complex motion sequences.

03

3D Digital Twin

Simulate robot movements in a 3D virtual environment before executing commands on real hardware — safe, repeatable, and educational.

04

Structured Course Modules

Embedded robotics curriculum covering theory, programming, and hands-on operation. No external LMS required — everything is in one place.

05

ROS 2 Communication Layer

Translates user programs into joint motion commands over USB serial via the Robot Operating System 2 — industry-standard protocol for robotics.

Next.js Blockly Python ROS 2 WebSockets

Future of the Project

EduBot is a foundation — here's where it can go from here.

01

Expanded Curriculum

Develop a full course library covering inverse kinematics, trajectory planning, and computer vision — structured for secondary school through university level.

02

Multi-Robot Coordination

Scale the platform to support multiple EduBots operating in parallel within a single classroom, enabling collaborative and competitive robotics exercises.

03

End-Effector Ecosystem

Standardise a quick-swap tool interface so students can swap grippers, cameras, and pen tools — turning EduBot into a true multi-application platform.

04

Open-Source Release

Publish all CAD files, PCB schematics, and platform source code under an open licence — letting institutions worldwide build, adapt, and improve EduBot.
