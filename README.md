# EduBot Portfolio

Portfolio site for **EduBot** — an affordable 5-axis robotic arm with integrated learning platform, built by Team EduBot at ICAM Strasbourg-Europe (2025–2026).

## Stack

- [Vite](https://vitejs.dev/) — build tool
- [GSAP](https://gsap.com/) — animations
- [Three.js](https://threejs.org/) — 3D visuals
- Vanilla HTML / CSS / JS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build   # outputs to /dist
npm run preview # preview the production build locally
```

## Deploy

Hosted on [Vercel](https://vercel.com). Every push to `main` triggers a new deployment.
